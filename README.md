# 🧠 Шпаргалка по математике (5–9 классы)

Полезные формулы и определения по алгебре и геометрии с пояснениями. Удобно для повторения, подготовки к ОГЭ, олимпиадам и печати.

## 📊 Алгебра

### 🧮 Степени и корни
- aᵐ ⋅ aⁿ = aᵐ⁺ⁿ (при умножении степеней с одинаковыми основаниями показатели складываются)
- aᵐ / aⁿ = aᵐ⁻ⁿ (при делении степеней с одинаковыми основаниями показатели вычитаются)
- (aᵐ)ⁿ = aᵐⁿ (при возведении степени в степень показатели перемножаются)
- (ab)ⁿ = aⁿbⁿ (степень произведения равна произведению степеней)
- (a/b)ⁿ = aⁿ/bⁿ (степень частного равна частному степеней)
- a⁻ⁿ = 1/aⁿ (отрицательная степень)
- a⁰ = 1 (a ≠ 0) (любое число в нулевой степени равно 1)
- √(a²) = |a| (квадратный корень из квадрата равен модулю)
- ⁿ√(aᵐ) = aᵐ/ⁿ (корень n-й степени можно представить как степень с дробным показателем)

### 🧩 Формулы сокращённого умножения
- (a + b)² = a² + 2ab + b² (квадрат суммы)
- (a - b)² = a² - 2ab + b² (квадрат разности)
- a² - b² = (a - b)(a + b) (разность квадратов)
- (a + b)³ = a³ + 3a²b + 3ab² + b³ (куб суммы)
- (a - b)³ = a³ - 3a²b + 3ab² - b³ (куб разности)
- a³ + b³ = (a + b)(a² - ab + b²) (сумма кубов)
- a³ - b³ = (a - b)(a² + ab + b²) (разность кубов)
- (a + b + c)² = a² + b² + c² + 2ab + 2ac + 2bc (квадрат трёхчлена)

### 📈 Уравнения и неравенства
- Линейное: ax + b = 0 ⇒ x = -b/a (где a ≠ 0)
- Квадратное: ax² + bx + c = 0 (где a ≠ 0)
  Дискриминант: D = b² - 4ac (определяет количество корней)
  Корни: x = (-b ± √D)/2a
  Теорема Виета: x₁ + x₂ = -b/a, x₁x₂ = c/a (связь корней с коэффициентами)

### 📐 Пропорции и проценты
- a/b = c/d ⇒ ad = bc (основное свойство пропорции)
- p% от a = (p/100) ⋅ a (найти процент от числа)
- a = (b ⋅ 100)/p (найти число по его проценту)
- Увеличение на p%: a ⋅ (1 + p/100) (число увеличилось на p%)
- Уменьшение на p%: a ⋅ (1 - p/100) (число уменьшилось на p%)
- Сложные проценты: A = P(1 + r/100)ⁿ (где P - начальная сумма, r - процентная ставка, n - период)

### 📊 Прогрессии
- Арифметическая прогрессия:
  aₙ = a₁ + (n - 1)d (n-й член прогрессии, где d - разность)
  Sₙ = n/2 ⋅ (a₁ + aₙ) = n/2 ⋅ [2a₁ + (n-1)d] (сумма n первых членов)
- Геометрическая прогрессия:
  bₙ = b₁ ⋅ qⁿ⁻¹ (n-й член прогрессии, где q - знаменатель)
  Sₙ = b₁ ⋅ (1 - qⁿ)/(1 - q) (q ≠ 1) (сумма n первых членов)
  S = b₁/(1 - q) (|q| < 1, бесконечная убывающая прогрессия)

### 📍 Координаты и графики
- Расстояние между точками: d = √[(x₂ - x₁)² + (y₂ - y₁)²] (теорема Пифагора на координатной плоскости)
- Середина отрезка: M = ((x₁ + x₂)/2, (y₁ + y₂)/2) (среднее арифметическое координат)
- Уравнение прямой: y = kx + b (где k - угловой коэффициент, b - свободный член)
- Угловой коэффициент: k = (y₂ - y₁)/(x₂ - x₁) (тангенс угла наклона прямой)
- Уравнение окружности: (x - a)² + (y - b)² = R² (где (a,b) - центр, R - радиус)

## 📐 Геометрия

### 🔺 Треугольники
- Периметр: P = a + b + c (сумма длин сторон)
- Площадь:
  S = 1/2 ⋅ a ⋅ h (половина произведения основания на высоту)
  S = 1/2 ⋅ ab ⋅ sinγ (половина произведения двух сторон на синус угла между ними)
  S = abc/4R (через стороны и радиус описанной окружности)
  S = pr (через полупериметр и радиус вписанной окружности)
- Формула Герона:
  p = (a + b + c)/2 (полупериметр)
  S = √[p(p - a)(p - b)(p - c)] (площадь через стороны)
- Теорема Пифагора: c² = a² + b² (в прямоугольном треугольнике квадрат гипотенузы равен сумме квадратов катетов)
- Теорема косинусов: c² = a² + b² - 2ab⋅cosγ (обобщение теоремы Пифагора)
- Теорема синусов: a/sinα = b/sinβ = c/sinγ = 2R (отношение стороны к синусу противолежащего угла равно диаметру описанной окружности)
- Медиана: mₐ = 1/2 ⋅ √(2b² + 2c² - a²) (длина медианы к стороне a)
- Биссектриса: lₐ = √[bc((b+c)² - a²)]/(b+c) (длина биссектрисы к стороне a)

### 📏 Углы и окружности
- Сумма углов треугольника: α + β + γ = 180° (в любом треугольнике)
- Смежные углы: α + β = 180° (углы, у которых одна сторона общая, а другие являются продолжениями друг друга)
- Вертикальные углы: равны (углы, образованные при пересечении двух прямых)
- Центральный угол равен дуге, на которую опирается
- Вписанный угол равен половине дуги, на которую опирается
- Окружность:
  C = 2πR (длина окружности)
  S = πR² (площадь круга)
- Сектор:
  S = (πR² ⋅ α)/360° (площадь сектора в градусах)
  S = 1/2 ⋅ R² ⋅ α (α в радианах) (площадь сектора в радианах)
- Дуга:
  l = (2πR ⋅ α)/360° (длина дуги в градусах)
  l = R ⋅ α (α в радианах) (длина дуги в радианах)

### 🔷 Четырёхугольники

| Фигура         | Формулы                            | Свойства |
|----------------|-------------------------------------|----------|
| Прямоугольник  | P = 2(a + b), S = a ⋅ b           | Все углы 90°, противоположные стороны равны |
| Квадрат        | P = 4a, S = a², d = a√2           | Все стороны и углы равны, диагонали равны и перпендикулярны |
| Параллелограмм | P = 2(a + b), S = a ⋅ h           | Противоположные стороны параллельны и равны |
| Ромб           | P = 4a, S = 1/2 ⋅ d₁ ⋅ d₂         | Все стороны равны, диагонали перпендикулярны |
| Трапеция       | P = a + b + c + d, S = 1/2(a + b) ⋅ h | Две стороны параллельны (основания) |

### 📦 Объёмные тела
- Куб:
  V = a³ (объём)
  S = 6a² (площадь полной поверхности)
  d = a√3 (диагональ куба)
- Параллелепипед:
  V = abc (объём)
  S = 2(ab + bc + ac) (площадь полной поверхности)
  d = √(a² + b² + c²) (диагональ параллелепипеда)
- Призма:
  V = Sосн ⋅ h (объём, где Sосн - площадь основания)
  Sбок = Pосн ⋅ h (площадь боковой поверхности, где Pосн - периметр основания)
- Пирамида:
  V = 1/3 ⋅ Sосн ⋅ h (объём)
- Цилиндр:
  V = πR²h (объём)
  Sбок = 2πRh (площадь боковой поверхности)
  Sполн = 2πR(h + R) (площадь полной поверхности)
- Конус:
  V = 1/3 ⋅ πR²h (объём)
  Sбок = πRl (площадь боковой поверхности)
  Sполн = πR(l + R) (площадь полной поверхности)
  l = √(R² + h²) (образующая конуса)
- Шар:
  V = 4/3 ⋅ πR³ (объём шара)
  S = 4πR² (площадь поверхности сферы)

## ⚖️ Теоремы и признаки

### Равенство треугольников
- По двум сторонам и углу между ними
- По стороне и двум прилежащим углам
- По трём сторонам

### Подобие треугольников
- По двум пропорциональным сторонам и углу между ними
- По двум равным углам
- По трём пропорциональным сторонам

### Параллельность и перпендикулярность
- Если накрест лежащие углы равны, то прямые параллельны
- Если соответственные углы равны, то прямые параллельны
- Если сумма односторонних углов равна 180°, то прямые параллельны
- Прямая перпендикулярна плоскости, если она перпендикулярна двум пересекающимся прямым в этой плоскости
